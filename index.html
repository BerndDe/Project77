Visual Studio Code is an App that allows you to code in html, css and js freely along with providing lots of add-ons that you can install. To Start a new project all you have to do is create a new folder and open it in VSC, add index.html and style.css, also js if you need and then start coding!


The console screen is mainly used for debugging purposes but also many other things. You open the console screen by right-clicking on your keyboard/mouse and going to inspect and selecting console in one the options at the top. To display something on the console screen all you have to do is create a variable with and at some text with it, then use console.log to displat it!
